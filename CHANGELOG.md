# Changelog

## [1.0.0] - 2026-02-04

### Added
- Initial release of Lyrah OS
- Luna Mode: gamescope-based gaming session with Luna UI
- Desktop Mode: Full KDE Plasma 6 desktop
- Luna UI: Qt6/QML gaming frontend with store backends (Steam, Epic/GOG via Heroic, Lutris, Custom)
- Session switching via lyrah-switch-mode
- Auto-login configuration via lyrah-configure-autologin
- GPU auto-detection and driver installation
- Update system with stable/testing/dev channels
- Log upload and crash monitoring
- SDDM space theme
- Plymouth boot splash
- Calamares installer configuration
- GitHub Actions CI/CD (ISO build, container images, testing, releases)
- Six built-in UI themes: Nebula Dark, Space Purple, Cyber Neon, AMOLED Black, Forest Green, Sunset Orange
